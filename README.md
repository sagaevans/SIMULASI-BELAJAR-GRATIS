# 📚 Simulasi Belajar Gratis

Platform latihan **TOEFL, Try Out, dan Psikotes** gratis berbasis web. Website ini berfungsi sebagai landing page utama yang menghubungkan ke berbagai simulasi belajar.

## 🌐 Demo

Kunjungi website ini di: [https://sagaevans.github.io/SIMULASI-BELAJAR-GRATIS](https://sagaevans.github.io/SIMULASI-BELAJAR-GRATIS)

## 📋 Fitur

- **TOEFL Simulation** — Latihan soal TOEFL (Listening, Structure, Reading)
- **Try Out Sekolah** — Simulasi ujian sekolah
- **Psikotes Online** — Latihan psikotes untuk persiapan kerja

## 🛠️ Teknologi

- HTML5
- CSS3 (responsive, no framework)
- Vanilla JavaScript

Tidak menggunakan framework apapun (React, Vue, Bootstrap, Tailwind, dsb.)

## 📁 Struktur File

```
SIMULASI-BELAJAR-GRATIS/
├── index.html      # Halaman utama
├── style.css       # Stylesheet
├── script.js       # JavaScript interaksi
└── README.md       # Dokumentasi
```

## 🚀 Cara Deploy di GitHub Pages

1. Buka repository ini di GitHub
2. Pergi ke **Settings** → **Pages**
3. Di bagian **Source**, pilih branch `main` dan folder `/ (root)`
4. Klik **Save**
5. Tunggu beberapa menit, website akan tersedia di URL GitHub Pages

## ✏️ Cara Mengganti Link

Untuk mengganti placeholder link pada kartu kategori:

1. Buka file `index.html`
2. Cari komentar `<!-- TODO: Ganti URL ... -->`
3. Ganti value `href` dengan URL yang sesuai

Contoh:
```html
<a href="https://sagaevans.github.io/TOEFL" class="btn btn--card" target="_blank">
    Mulai Latihan
</a>
```
